## `radical()`

Returns a array which represents the simplified square root of a number. In other words, the returned array `[a, b]` represents the expression `a&radic;b`.

```javascript
Calc.radical(9);
// [3,1]
Calc.radical(3);
// [1,3]
Calc.radical(-20);
// [2,-5]
```
