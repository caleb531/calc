## `nearest()`

Returns a number rounded to the nearest multiple of `n`.

### Round 5 to the nearest multiple of 2
```javascript
Calc.nearest(5, 2);
// 6
```

### Round 24 to the nearest multiple of 10
```javascript
Calc.nearest(24, 10);
// 20
```

### Round 2.7 to the nearest multiple of 0.5
```javascript
Calc.nearest(2.7, 0.5);
// 2.5
```
