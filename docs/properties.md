## Properties

```javascript
Calc.PI;
// 3.141592653589793
```

### Euler's Number *e*
```javascript
Calc.E;
// 2.718281828459045
```

### Golden Ratio *&Phi;*
```javascript
Calc.PHI;
// 1.618033988749895
```

### Gravitational Constant *G*
```javascript
Calc.G;
// 6.673848e-11
```
